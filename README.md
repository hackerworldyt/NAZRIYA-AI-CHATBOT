## Installation

```sh
$ git clone https://github.com/MR-JINN-OF-TG/NAZRIYA-AI-CHATBOT
$ cd NAZRIYA-AI-CHATBOT
$ pip3 install -U -r requirements.txt
$ cp sample_config.py config.py
```
Edit `config.py` with your own values.
```sh
$ python3 nazriya.py
